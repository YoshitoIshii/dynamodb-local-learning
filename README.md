# dynamodb-local-learning

learning dynamodb local personally.

## Usage
  
* listTables  
    `./localdynamo tables`
* describeTable  
    `./localdynamo desc <table-name>`
* scan   
    `./localdynamo scan <table-name> [<column>[,<column> ... ]]`
* scan and filter by specified id  
    `./localdynamo scanidf <table-name> <id>`
